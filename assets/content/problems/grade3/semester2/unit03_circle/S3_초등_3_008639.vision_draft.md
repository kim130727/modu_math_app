# [전체적인 구성]
- 화면 상단 왼쪽에 문제 번호와 지시문이 한 줄로 배치되어 있음.
- 중앙에는 원 그림이 크게 있고, 원의 중심에서 바깥 둘레로 뻗는 선분 3개가 보임.
- 원 아래에는 연두색 테두리의 큰 직사각형 박스가 있으며, 그 안에 선택형 문장이 들어 있음.
- 박스 아래 왼쪽에는 `(정답)` 표시와 함께 정답 문장, 그 아래에 `(해설)` 표시와 함께 해설 문장이 세로로 배치되어 있음.
- 전체적으로 “그림 보고 괄호 안 선택” 형태의 초등 수학 문제 구성임.

# [보이는 텍스트]
- `11. 위에 반지름을 3개 그은 것입니다. 반지름을 모두 적당한 말을 선택하세요.`
- 박스 안 문장:
  - `한 원에서 원의 반지름들은 모두 ( 같습니다, 다릅니다 ).`
- 하단 텍스트:
  - `(정답)같습니다`
  - `(해설)반지름은 모두 같습니다.`
- 원 내부에는 글자 `O`처럼 보이는 작은 표시가 중심 부근에 있음.
- 중심점처럼 보이는 위치에 작은 분홍색 점이 있음.

# [반복되는 시각 패턴]
- 원의 중심에서 바깥쪽으로 뻗는 선분이 3개 반복적으로 나타남.
- 세 선분은 서로 다른 방향으로 뻗어 있으며 색이 각각 다름:
  - 왼쪽 방향에 가까운 하늘색 선
  - 오른쪽 위 방향의 연보라색 선
  - 아래 방향의 초록색 선
- 모두 원의 중심점 근처에서 시작해 원 둘레까지 이어지는 형태로 보이며, 이는 같은 종류의 선분을 여러 개 보여 주는 시각적 패턴임.
- 박스 안의 선택지 구조도 반복형 패턴으로 볼 수 있음: 한 문장 안에 두 개의 괄호 선택지 `( 같습니다, 다릅니다 )`가 나열됨.

# [세부 내용]
- 문제 번호 `11`은 좌상단에 있고, 그 오른쪽에 설명 문장이 이어짐.
- 원은 화면 중앙보다 약간 위쪽에 위치하며, 외곽선은 얇은 회색/검정색 계열로 보임.
- 원 중심 부근에는 작은 흰색/빈 원 표시와 분홍색 점이 함께 있어 중심을 강조하는 듯함.
- 세 선분은 모두 중심점에서 시작하며, 원의 경계까지 도달하는 길이로 그려져 있음.
- 선분 배치:
  - 왼쪽으로 거의 수평에 가까운 하늘색 선분
  - 오른쪽 위로 올라가는 보라색 선분
  - 아래쪽으로 거의 수직에 가까운 초록색 선분
- 연두색 테두리 박스는 원 아래에 가로로 길게 놓여 있으며, 내부 문장은 가운데 정렬된 형태로 보임.
- 박스 안에는 보기로서 `( 같습니다, 다릅니다 )`가 제시되어 있어 둘 중 하나를 고르는 형태임.
- 하단의 `(정답)`과 `(해설)`은 굵지 않은 일반 텍스트처럼 보이며, 답안 예시가 시각적으로 제공되어 있음.

# [Python DSL 작성 힌트]
- 원 도형을 1개 그린 뒤, 중심에서 반지름 역할의 선분 3개를 서로 다른 방향으로 추가하는 구조로 작성하면 됨.
- 중심점은 작은 점 또는 작은 원으로 표시하고, 그 위치에서 3개 선분이 시작되도록 좌표를 맞추는 것이 중요함.
- 텍스트는 상단 문제 문장, 원 아래의 선택 문장, 하단의 `(정답)`, `(해설)` 순서로 배치하면 됨.
- 선택 문장 박스는 연두색 테두리의 직사각형으로 표현하고, 내부 문장은 가운데 정렬로 배치하는 방식이 적절함.
- 정답/해설 문구는 별도 텍스트 블록으로 아래쪽 왼편에 정렬하는 구성이 자연스러움.
- 그림의 선분 색을 각각 다르게 지정하면 원본 시각과 비슷해짐.

# [주의할 점]
- 문제를 수학적으로 풀어 답을 새로 계산하거나 추론하지 말아야 함.
- 박스 안의 괄호 선택지 중 무엇이 정답인지 새로 판단해서 적지 말고, 보이는 텍스트만 기록해야 함.
- 원의 중심과 분홍색 점의 정확한 의미는 이미지상 완전히 명시되어 있지 않으므로 단정하지 말 것.
- `O`처럼 보이는 표시는 문자인지 점 표시인지 애매하므로, 관찰된 형태만 설명하는 것이 안전함.
- 선분이 정확히 원의 반지름인지 여부는 문제 문장에 의존하므로, 그림만으로 추가 해석하지 말 것.

# [불확실한 부분]
- 원 내부의 작은 `O` 모양 표시는 글자인지 중심 표시인지 이미지상 애매함.
- 분홍색 점이 정확히 중심점인지, 또는 강조 표시인지 완전히 확정할 수 없음.
- 선분의 정확한 끝점이 원의 테두리와 완전히 일치하는지 일부는 해상도상 모호함.
- 박스 안 문장의 글꼴 굵기와 정확한 자간은 이미지 해상도 때문에 세부적으로 단정하기 어려움.
