[문제 개요]
- 문제 번호: `37.`
- 질문 문장: `원의 지름은 어느 선분인가요?`
- 형태: 원 그림을 보고 보기 4개 중 알맞은 선분을 고르는 객관식 문제
- 하단에 `(정답)②`, `(해설)중심을 지나는 선분 ㄷㄹ이 원의 지름입니다.`가 인쇄되어 보임
- 원본 이미지 기준으로, 원의 지름 후보가 그림 안 여러 가로선분으로 제시되어 있음

[원본에 보이는 요소]
- 좌상단에 문제 번호와 질문 문장이 있음
- 우상단에 큰 원 도형이 배치되어 있음
- 원 내부에 수평에 가까운 가로 선분이 4개 보임
- 각 선분의 양 끝에 한글 자모 표기가 붙어 있음
  - 맨 위 선분: 왼쪽 `ㄱ`, 오른쪽 `ㄴ`
  - 두 번째 선분: 왼쪽 `ㄷ`, 오른쪽 `ㄹ`
  - 세 번째 선분: 왼쪽 `ㅁ`, 오른쪽 `ㅂ`
  - 네 번째 선분: 왼쪽 `ㅅ`, 오른쪽 `ㅈ`
- 원 내부 중앙 부근에 작은 동그라미 표기 `ㅇ`처럼 보이는 요소와, 그 아래 작은 분홍색 점이 있음
- 보기 4개가 아래쪽에 2열로 배치되어 있음
  - `① 선분 ㄱㄴ`
  - `② 선분 ㄷㄹ`
  - `③ 선분 ㅁㅂ`
  - `④ 선분 ㅅㅈ`
- 좌하단에 정답 표기와 해설 문장이 작게 따로 있음
- 문제 번호 앞에 작은 네모 모양 기호가 보임

[수식/관계 구조]
- 계산식이나 등식은 없음
- 관계 구조만 존재함:
  - 여러 선분 중 하나가 원의 중심을 지나는 선분으로 제시됨
  - 보기들은 모두 `선분 + 양 끝 문자 2개` 형식
  - 그림의 선분과 보기의 선분 표기가 대응되는 구조
- 의미적으로는 “어느 선분이 원의 지름인가?”를 묻는 분류형 문제
- 그림에서 중심을 지나는 가로 선분이 강조된 위치에 있음

[등호별 변환 의미]
- 해당 없음
- 등호 연쇄나 식 변형이 보이지 않음

[빈칸 정책]
- 빈칸형 문제가 아님
- 답 칸은 따로 만들지 않음
- 원본에 정답 `②`가 인쇄되어 보이지만, 이를 추론한 값으로 채워 넣지 말고 관찰 사실로만 별도 표기함
- 빈칸이 존재하지 않으므로 추정값 삽입 금지 원칙만 명시하면 충분함

[반복 패턴]
- 보기 번호 `①, ②, ③, ④`가 원형 번호 표시로 반복됨
- 보기 텍스트는 모두 `선분` 뒤에 한글 자모 두 글자가 오는 동일한 형식임
- 도형 내부에는 좌우 대응되는 자모 쌍이 반복 배치됨
- 수평 선분들이 위에서 아래로 여러 개 평행하게 배열됨
- 중앙을 지나는 선분이 다른 선분들과 비교될 수 있도록 위치상 중요하게 배치됨

[DSL 작성 힌트]
- 전체 레이아웃은 상단 좌측의 문제 문장, 상단 우측의 원 도형, 하단의 보기 2열, 최하단의 정답/해설 영역으로 나누어 생각하면 좋음
- 원 도형은 큰 원 1개와 그 내부의 수평 선분 4개로 구성
- 각 선분의 양 끝에는 자모 라벨을 정확히 대응시켜 배치
  - 1번째: `ㄱ`-`ㄴ`
  - 2번째: `ㄷ`-`ㄹ`
  - 3번째: `ㅁ`-`ㅂ`
  - 4번째: `ㅅ`-`ㅈ`
- 두 번째 선분 근처 중앙에는 작은 원형 표시와 분홍색 점이 함께 보이도록 배치
- 보기의 번호 원형 표시는 파란색 계열로 보이며, 글자보다 약간 작게 처리하면 유사함
- 보기 2열 배치는 좌측 열에 `①`, `③`, 우측 열에 `②`, `④`가 놓인 형태로 보임
- 정답/해설은 본문과 분리된 작은 글씨 영역으로 처리
- 문제 번호 앞의 작은 네모 기호는 필요시 장식 요소로만 반영 가능

[semantic/solvable 힌트]
- 이 문제는 도형 판별형 객관식 문제로 보임
- 그림과 해설 문장에 따르면, 중심을 지나는 선분이 핵심 판별 기준임
- 원의 지름은 개념상 중심을 지나는 현에 해당하는 선분이라는 의미로 읽힘
- 다만 정답을 계산해서 추론할 필요는 없고, 원본에 이미 `②`가 보이는 사실만 기록하면 충분함
- “중심을 지나는 선분”이라는 해설 문구는 그림의 두 번째 가로선분을 가리키는 설명으로 보임

[주의 및 TODO]
- `②`는 원본에 인쇄되어 보이는 정답 표기이므로 관찰 사실로만 남기고, 풀이 결과처럼 재도출하지 말 것
- 빈칸이 없는 문제이므로 답안 공란을 만들 필요 없음
- 원 내부 중앙의 `ㅇ`처럼 보이는 동그라미와 분홍 점의 정확한 의미는 불명확하므로 TODO로 남김
- 문제 번호 앞 작은 네모 기호의 역할은 불확실하므로 장식/편집표시 여부를 단정하지 말 것
- 선택지 글꼴, 자간, 세로 간격은 이미지 기준으로 대략적인 배치만 맞추면 됨
- 도형의 선 굵기와 자모 위치는 약간의 오차를 허용하되, 좌우 대응과 상하 순서는 유지해야 함
