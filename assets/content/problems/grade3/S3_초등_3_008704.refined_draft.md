[문제 개요]
- 초등 수학 문제 화면이다.
- 주제는 **원의 중심과 반지름의 규칙**을 찾는 내용이다.
- 왼쪽에는 격자 위에 그려진 **원 3개**가 보이고, 오른쪽에는 말풍선 설명과 캐릭터 **승아**가 있다.
- 아래에는 연한 초록색 배경의 **선택 문장 영역**이 있으며, 맨 아래에는 **정답**과 **해설** 텍스트가 보인다.

[원본에 보이는 요소]
- 상단 문제 문장:
  - `□ 98. 승아는 규칙에 따라 원을 그렸습니다. 승아가 원을 그린 규칙을 찾아 알맞은 것을 선택해보세요.`
- 왼쪽 그림:
  - 파란 점선 격자 배경
  - 원 3개가 겹쳐 있는 그림
  - 중심을 나타내는 작은 분홍색 점
- 오른쪽 말풍선:
  - `원의 중심이 같고`
  - `원의 반지름은 다르게 하여`
  - `그린 거야.`
- 캐릭터 이름표:
  - `승아`
- 연한 초록색 선택 문장 영역:
  - `원의 중심이 ( 같습니다, 다릅니다 ).`
  - `원의 반지름이 모두 ( 1, 2, 3 )칸씩 늘어났습니다.`
- 하단 정답 표기:
  - `(정답) 같습니다, 1`
- 하단 해설:
  - `(해설) 원의 중심이 같습니다.`
  - `원의 반지름이 모두 1칸씩 늘어났습니다.`

[수식/관계 구조]
- 이 화면에는 등호를 이용한 계산식 변형은 보이지 않는다.
- 대신 다음과 같은 **관계 선택형 문장 구조**가 보인다.
  - 원의 중심: `( 같습니다 / 다릅니다 )`
  - 반지름 증가량: `( 1 / 2 / 3 )칸씩`
- 왼쪽의 3개의 원은 **동심원처럼 같은 중심을 공유하는 구성**으로 보인다.
- 반지름은 바깥쪽으로 갈수록 커지는 형태이다.

[등호별 변환 의미]
- 해당 없음.
- 화면에 `=`를 포함한 식이나 단계적 변형이 보이지 않는다.
- 따라서 등호마다 무엇이 어떻게 바뀌는지 설명할 대상이 없다.

[빈칸 정책]
- 빈칸이나 선택형 괄호 안에는 **추론값을 넣지 않는다.**
- 이미지에 실제로 보이는 선택지는 그대로 유지한다.
- 정답이 하단에 인쇄되어 보이더라도, **빈칸을 그 값으로 미리 채우지 않는다.**
- DSL 작성 시에도 빈칸은 빈칸 상태로 표현하고, 보이는 선택지 텍스트만 보존한다.

[반복 패턴]
- 왼쪽 그림에서 원 3개가 반복적으로 겹쳐 나타난다.
- 파란 점선 격자 칸이 규칙적으로 반복되어 배경을 이룬다.
- 선택 문장 영역에서는 같은 문장 틀 안에 괄호 선택지가 반복된다.
  - `원의 중심이 ( ... ).`
  - `원의 반지름이 모두 ( ... )칸씩 늘어났습니다.`
- 해설 문장도 두 줄 구조로 반복되어 배치된다.

[DSL 작성 힌트]
- 화면을 다음 영역으로 나누어 처리하면 좋다.
  1. 상단 문제 문장
  2. 왼쪽 격자+원 그림
  3. 오른쪽 말풍선+승아 캐릭터
  4. 가운데 아래 연두색 선택 문장 박스
  5. 맨 아래 정답/해설 텍스트
- 왼쪽 그림:
  - 파란 점선 격자 배경을 별도 요소로 둔다.
  - 원 3개는 같은 중심을 가진 중첩 원으로 배치한다.
  - 중심점은 작은 분홍색 점으로 표시한다.
- 오른쪽 말풍선:
  - 타원형 말풍선과 아래 꼬리를 가진 구조로 표현한다.
  - 말풍선 안 문장은 3줄로 보이게 배치한다.
- 연한 초록색 박스:
  - 직사각형 배경 박스로 처리한다.
  - 내부 텍스트는 2줄로 정렬한다.
  - 괄호 안 선택지는 텍스트 그대로 둔다.
- 하단 정답/해설:
  - `(정답)`과 `(해설)` 라벨이 앞에 붙은 텍스트 블록으로 처리한다.
  - 정답 표기는 관찰 사실로만 기록하고, 빈칸 채우기용으로 사용하지 않는다.
- 레이아웃상 오른쪽 캐릭터와 말풍선은 왼쪽 그림보다 위쪽에 배치되어 있다.
- 상단 문장과 본문 그림 사이에는 충분한 여백이 있다.

[semantic/solvable 힌트]
- 이 문제는 **그림의 규칙을 읽고 선택지를 고르는 문제**이다.
- 핵심 의미는 다음 두 가지이다.
  - 원의 중심이 같은지 여부
  - 반지름이 얼마나 늘어났는지
- 이미지에 보이는 해설 텍스트에 따르면:
  - 원의 중심이 같다고 되어 있다.
  - 원의 반지름이 모두 1칸씩 늘어났다고 되어 있다.
- 다만, 이 값들은 **관찰 사실**로만 취급하고, 추가 추론은 하지 않는다.
