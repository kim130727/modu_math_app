[문제 개요]
- 문제 번호 `27.`이 상단 왼쪽에 보인다.
- 본문 지시문은 `규칙에 따라 원을 그린 것입니다. 그린 규칙을 바르게 말한 사람을 선택해 보세요.`이다.
- 왼쪽에는 모눈 격자 위에 원 3개와 중심점 1개가 그려져 있다.
- 오른쪽에는 학생 2명(승아, 재원)과 각각의 말풍선이 배치되어 있다.
- 하단에는 `(정답)재원`, `(해설)...` 문장이 인쇄되어 있다.

[원본에 보이는 요소]
- 상단 문장:
  - `27. 규칙에 따라 원을 그린 것입니다. 그린 규칙을 바르게 말한 사람을 선택해 보세요.`
- 왼쪽 그림:
  - 하늘색 모눈 격자 배경의 정사각형 영역
  - 중심점처럼 보이는 분홍색 점 1개
  - 그 점을 중심으로 보이는 원 3개
- 오른쪽 위 학생:
  - 이름표 `승아`
  - 말풍선 텍스트:
    - `원의 중심이 모두`
    - `다르고 원의 반지름이`
    - `모눈 1칸씩 늘어나게`
    - `그렸어요.`
- 오른쪽 아래 학생:
  - 이름표 `재원`
  - 말풍선 텍스트:
    - `원의 중심이 모두`
    - `같고 원의 반지름이`
    - `모눈 2칸씩 늘어나게`
    - `그렸어요.`
- 하단 정답 표시:
  - `(정답)재원`
- 하단 해설:
  - `(해설)원의 중심이 모두 같고 원의 반지름이 각각 모눈 1칸, 3칸, 5칸으로 2칸씩 늘어나는 규칙입니다.`

[수식/관계 구조]
- 이 문항에는 등호로 연결된 수식 변형은 보이지 않는다.
- 보이는 관계는 수식 계산이 아니라 그림의 규칙 설명이다.
- 추론되는 수학적 의미:
  - 원 3개는 같은 중심을 공유하는 동심원 구조로 보인다.
  - 원의 크기는 바깥으로 갈수록 커진다.
  - 해설 문장에는 반지름이 `1칸, 3칸, 5칸`으로 늘어난다고 적혀 있다.
- 단, 반지름의 정확한 칸수 판독은 이미지 해상도에 의존하므로, DSL에서는 텍스트로 처리하거나 도형 배치 기준으로만 반영하는 것이 안전하다.

[등호별 변환 의미]
- 해당 없음.
- 등호 연쇄나 계산식 변형이 제시되지 않았다.

[빈칸 정책]
- 빈칸 문제는 아니다.
- 원본에 인쇄된 말풍선 문장과 하단의 정답/해설은 보이는 그대로 옮긴다.
- 추론한 값으로 빈칸을 채우지 않는다.
- 만약 DSL에서 말풍선 줄바꿈이나 배치를 다시 잡더라도, 텍스트 내용 자체는 원문 관찰값만 사용한다.

[반복 패턴]
- 왼쪽 그림에서 원 3개가 반복적으로 나타난다.
- 세 원은 같은 중심을 공유하는 반복 구조로 보인다.
- 모눈 격자 선이 일정 간격으로 반복된다.
- 오른쪽 인물 영역은 위아래로 반복되는 2단 구조이다.
  - 위: 승아 + 말풍선
  - 아래: 재원 + 말풍선
- 두 말풍선은 모두 타원형이며, 왼쪽을 향한 꼬리가 있다.
- 말풍선 내부는 여러 줄 텍스트로 반복적인 줄바꿈 형태를 가진다.

[DSL 작성 힌트]
- 페이지를 크게 다음 4영역으로 나누면 좋다.
  1. 상단 문항 문장
  2. 왼쪽 모눈 격자 + 원 3개 + 중심점
  3. 오른쪽 학생 2명 + 이름표 + 말풍선 2개
  4. 하단 정답/해설 문장
- 왼쪽 그림 렌더링 힌트:
  - 모눈 격자는 정사각형 영역 안에 배치
  - 가는 점선/실선 형태의 격자
  - 중심점 1개를 중앙 근처에 작은 분홍색 점으로 표시
  - 동심원 3개를 같은 중심으로 정렬
- 오른쪽 인물/말풍선 렌더링 힌트:
  - 승아는 위, 재원은 아래에 세로로 배치
  - 각 학생의 오른쪽에 말풍선 배치
  - 말풍선은 타원형 외곽선 + 왼쪽 꼬리
  - 이름표는 학생 아래쪽의 둥근 사각형 배지처럼 보이게 처리
- 텍스트 배치 힌트:
  - 상단 문항은 2줄로 줄바꿈될 수 있으므로 자연스럽게 래핑
  - 말풍선 내부 텍스트는 3~4줄 정도로 중앙 정렬에 가깝게 배치
  - 하단 정답/해설은 본문과 분리된 보조 텍스트 영역으로 둔다
- 도형 분리 힌트:
  - 격자 배경
  - 중심점
  - 원 3개
  - 학생 얼굴/몸
  - 이름표
  - 말풍선
  - 말풍선 텍스트
  를 별도 객체로 분해하면 DSL 작성이 수월하다.

[semantic/solvable 힌트]
- 이 문항의 핵심 의미는 “그림의 생성 규칙을 말하는 사람을 고르는 것”이다.
- 보이는 해설 문장에 따르면, 원들은 같은 중심을 가지며 반지름이 일정한 간격으로 커지는 구조이다.
- 따라서 그림과 말풍선의 내용 비교가 핵심이지만, 정답을 새로 추론하지 말고 원본의 `(정답)재원` 표기를 관찰 사실로만 기록하면 된다.
- 수학적으로는 동심원과 반지름 증가 규칙을 읽는 문제로 해석된다.
