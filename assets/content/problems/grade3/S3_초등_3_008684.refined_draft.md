[문제 개요]
- 초등 수학 객관식 문항 1개.
- 주제는 **누름못과 띠 종이로 원을 그릴 때, 원 위의 점들과 누름못 사이의 거리 관계**를 묻는 그림 보고 판단하는 문제.
- 화면 구성은 상단 문제 문장, 가운데 그림 2개, 하단 주황색 선택 문장 박스, 맨 아래 정답/해설 표시로 이루어짐.
- 원본 이미지 하단에 정답과 해설이 **인쇄되어 보임**.

[원본에 보이는 요소]
- 상단 문제문:
  - `72. 누름 못과 띠 종이를 이용하여 원을 그렸습니다. 그림을 보고 알맞은 말을 선택하세요.`
- 가운데 왼쪽 그림:
  - 정사각형 외곽선 안에 큰 원.
  - 손, 연필, 띠 종이, 누름못이 함께 있는 과정 설명 그림.
  - 초록색 설명 문장과 `①`, `②` 번호가 보임.
- 가운데 오른쪽 그림:
  - 정사각형 외곽선 안에 원.
  - 중심의 작은 점.
  - 세로 점선과 가로 점선이 십자 형태로 교차.
  - 원 위에 4개의 검은 점이 표시됨.
- 하단 주황색 박스:
  - `누름 못이 꽂힌 점에서 원 위에 찍은 4개의 점까지의 길이는`
  - `모두 ( 같습니다, 다릅니다 ).`
- 맨 아래 정답 표시:
  - `(정답) 같습니다`
- 맨 아래 해설:
  - `(해설) 누름 못이 꽂힌 점에서 원 위에 찍은 4개의 점까지의 길이는 모두 같습니다.`

[수식/관계 구조]
- 이 문항에는 계산식, 분수, 등호 연쇄 변형이 없음.
- 핵심 관계는 **누름못이 꽂힌 점을 중심으로 한 원의 성질**을 말하는 문장형 선택지 구조임.
- 보이는 관계를 구조적으로 쓰면:
  - `누름못이 꽂힌 점` → 원의 중심 역할
  - `원 위의 4개 점` → 중심에서의 거리가 같은 대상
  - 선택어: `같습니다 / 다릅니다`
- 오른쪽 그림의 네 점은 원의 상하좌우 위치에 대칭적으로 배치되어 있어, 중심에서 같은 거리임을 시각적으로 보여줌.

[등호별 변환 의미]
- 등호가 있는 변형식은 보이지 않음.
- 따라서 단계별 수식 변환 설명은 해당 없음.
- 필요 시 DSL에서는 텍스트 문장만 독립 요소로 처리하면 됨.

[빈칸 정책]
- 괄호 안 선택지는 **빈칸을 채우는 것이 아니라, 원문에 이미 보이는 선택어를 그대로 유지**하는 형태로 다루어야 함.
- 추론해서 정답만 남기지 말고, 원문에 보이는 `같습니다, 다릅니다` 둘 다 유지해야 함.
- 원본에 정답이 보이더라도, 본문 DSL 작성에서는 **문항 본문과 정답 표시를 분리**해서 처리하는 것이 안전함.
- 빈칸이나 누락된 글자를 임의로 보정해 넣지 말 것.

[반복 패턴]
- 왼쪽 그림에는 과정 설명이 2단계로 반복됨:
  - `①` 한쪽 구멍에 누름못을 꽂아 띠 종이를 고정
  - `②` 다른 구멍에 연필을 꽂고 한 바퀴 돌려 원을 그림
- 오른쪽 그림에는 원 위 점이 4개 반복적으로 배치됨.
- 초록색 곡선 화살표가 왼쪽 그림에서 단계 진행을 연결하는 시각 패턴으로 보임.
- 가운데 두 그림은 같은 높이로 나란히 배치되어 비교 구조를 이룸.
- 하단 선택 문장은 박스 안에서 중앙 정렬에 가깝게 배치됨.

[DSL 작성 힌트]
- 전체를 다음 4구역으로 나누어 생각하면 좋음:
  1. 상단 문제문
  2. 가운데 좌우 그림 2개
  3. 하단 주황색 박스의 선택 문장
  4. 맨 아래 정답/해설 텍스트
- 왼쪽 그림 렌더링 힌트:
  - 정사각형 외곽선
  - 내부 원
  - 손 2개, 연필 1개, 띠 종이 1개, 누름못 1개
  - 초록색 단계 번호 `①`, `②`
  - 초록색 설명 문장 줄바꿈 배치
  - 곡선 화살표 2개
- 오른쪽 그림 렌더링 힌트:
  - 정사각형 외곽선
  - 내부 원
  - 중심점
  - 세로 점선
  - 가로 점선
  - 원 위 4개의 점
  - 중심 기준 십자 형태 정렬
- 주황색 박스:
  - 테두리 사각형
  - 내부 텍스트는 중앙 정렬
  - 괄호 안 선택어는 쉼표로 나란히 표현
- 정답/해설:
  - 본문과 분리하여 하단 작은 글씨로 배치
  - `(정답)`과 `(해설)` 라벨을 유지

[semantic/solvable 힌트]
- 보이는 사실:
  - 누름못이 꽂힌 점을 기준으로 원이 그려짐.
  - 원 위의 4개 점은 중심에서 같은 반지름 위에 있음.
  - 오른쪽 도식은 중심과 원 위 점들의 대칭성을 강조함.
- 추론되는 수학적 의미:
  - 중심에서 원 위의 임의의 점까지 거리는 같다.
- 다만, 문제를 풀어 답을 재구성하는 방식은 지양하고, 이미 보이는 문장과 도식 관계만 서술할 것.
- 원본 하단에 정답 `같습니다`가 보이지만, DSL 작성용 문맥에서는 **관찰 사실**로만 다루고, 본문 선택 구조를 유지하는 것이 좋음.
