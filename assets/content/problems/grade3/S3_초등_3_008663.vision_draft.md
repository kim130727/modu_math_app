# [전체적인 구성]
- 화면 상단에는 문제 번호와 문장이 왼쪽 정렬로 제시되어 있습니다.
- 본문에는 **컴퍼스가 눈금자 위에 놓인 그림 선택지 3개**가 가로 방향으로 배치되어 있습니다.
- 왼쪽 아래에는 **정답 표시가 붙은 작은 그림**이 따로 제시되어 있고, 그 아래에 해설 문장이 있습니다.
- 전체적으로는 **문장형 지시 + 그림 선택지 + 정답 예시/해설** 구조입니다.

# [보이는 텍스트]
- `□ 38.`
- `다음 중 반지름이 3 cm인 원을 그릴 수 있도록 컴퍼스를 바르게 벌린 것을 선택하세요.`
- `(정답)`
- `(해설)컴퍼스를 원의 반지름만큼 벌린 것을 찾습니다.`
- 눈금자에는 숫자 `0, 1, 2, 3`이 보입니다.
- 각 그림마다 컴퍼스 다리와 연필심/바늘 끝이 눈금자 위에 놓인 모습이 보입니다.

# [반복되는 시각 패턴]
- 같은 형태의 **컴퍼스 + 눈금자** 그림이 여러 개 반복됩니다.
- 각 선택지는 모두:
  - 아래쪽에 파란색 눈금자
  - 위쪽에 회색 컴퍼스 본체
  - 한쪽 다리는 뾰족한 바늘
  - 다른 쪽 다리는 주황색 연필심/표시 도구
- 눈금자 위에서 컴퍼스 다리의 벌어진 정도가 서로 다르게 표현되어 있습니다.
- 왼쪽 아래의 정답 그림도 같은 패턴을 따르며, 다른 선택지와 비교 대상으로 보입니다.

# [수식 단계] (해당될 때)
- 이 문제는 수식 변형 단계가 제시된 유형은 아닙니다.
- 등호 `=`로 이어지는 계산식은 보이지 않습니다.

# [등호별 변환 의미] (해당될 때)
- 해당 없음.  
- 식 변형이나 등호 단계가 없는 그림 선택 문제입니다.

# [세부 내용]
- 문제 문장은 “반지름이 **3 cm**인 원을 그릴 수 있도록 컴퍼스를 **바르게 벌린 것**”을 고르는 내용입니다.
- 그림 속 눈금자는 `0~3`까지 표시되어 있고, 컴퍼스의 양 끝이 눈금자 위의 서로 다른 위치에 놓여 있습니다.
- 첫 번째 선택지:
  - 컴퍼스의 한 끝은 0 부근,
  - 다른 끝은 1보다 조금 전/근처로 보입니다.
  - 벌어진 정도가 비교적 좁습니다.
- 두 번째 선택지:
  - 한 끝은 0 부근,
  - 다른 끝은 2 근처로 보입니다.
  - 첫 번째보다 더 벌어져 있습니다.
- 세 번째 선택지:
  - 한 끝은 0 부근,
  - 다른 끝은 3 근처로 보입니다.
  - 세 선택지 중 가장 크게 벌어진 모습으로 보입니다.
- 왼쪽 아래의 `(정답)` 표시가 있는 그림도 눈금자와 컴퍼스가 함께 그려져 있으며, 눈에 띄게 선택지와 유사한 형식입니다.
- 해설 문장은 컴퍼스의 벌림이 원의 반지름과 같아야 한다는 점을 말하고 있습니다.

# [Python DSL 작성 힌트]
- 선택형 문제이므로 Python DSL에서는 다음 요소를 분리해서 그리면 좋습니다.
  - 문제 번호와 지시문 텍스트
  - 선택지 3개의 컴퍼스-눈금자 그림
  - 정답 표시가 붙은 별도 예시 그림
  - 하단 해설 문장
- 각 그림은 동일한 기본 템플릿으로 만들고, **컴퍼스 벌어진 각도/다리 간 거리**만 다르게 설정하면 좋습니다.
- 눈금자는 파란 직사각형 바탕 위에 검은 눈금과 숫자를 넣는 구조로 표현하면 유사합니다.
- 컴퍼스는 회색 몸체, 바늘 끝은 검은색, 연필심 부분은 주황색으로 구분하면 좋습니다.
- 정답 표시 `(정답)`은 그림 왼쪽 위 또는 왼쪽 옆에 작은 텍스트로 배치되어 있습니다.
- 해설은 하단에 괄호가 포함된 문장으로 따로 배치하면 됩니다.

# [주의할 점]
- 정답을 직접 서술하지 말고, **그림 간 차이**만 설명해야 합니다.
- `3 cm`를 기준으로 어느 선택지가 맞는지 이 초안에서 결론 내리지 마세요.
- 눈금 위치는 확대 정도 때문에 약간 애매해 보일 수 있으므로, 단정적인 수치 해석은 피하는 것이 좋습니다.
- 그림의 정확한 비율보다 **상대적인 벌림 정도**를 중심으로 기록하는 것이 안전합니다.
- 정답 그림이 선택지 번호와 직접 연결되어 보이지만, 이 초안에서는 연결 관계를 단정하지 않고 “정답 표시가 있는 그림”으로만 적는 것이 좋습니다.

# [불확실한 부분]
- 각 선택지에서 컴퍼스 끝이 **정확히 어느 눈금에 닿는지**는 이미지 해상도상 다소 흐릿합니다.
- 왼쪽 아래의 `(정답)` 그림이 실제로 어떤 번호의 선택지와 대응하는지, 이 초안만으로는 확정하지 않는 것이 좋습니다.
- 눈금자 숫자와 컴퍼스 끝의 정확한 정렬은 약간 오차가 있어 보일 수 있습니다.
- 해설 문장은 읽을 수 있지만, 문장 끝의 띄어쓰기나 세부 표기는 이미지 품질에 따라 약간 달라질 수 있습니다.
