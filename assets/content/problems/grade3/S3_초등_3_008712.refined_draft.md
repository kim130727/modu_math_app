[문제 개요]
- 한 문제짜리 초등 수학 객관식/선택형 문제이다.
- 주제는 **원에서 지름이 되는 선분 찾기**이다.
- 화면 구성은 위쪽 문제 문장, 가운데 원 그림, 아래쪽 보기, 더 아래 정답 표기와 해설 문장 순서로 보인다.

[원본에 보이는 요소]
- 문제 번호와 문장:
  - `10. 원에서 원의 지름을 나타내는 선분을 선택하세요.`
- 그림:
  - 중심이 표시된 원 1개가 보인다.
  - 원 둘레와 내부에 점/문자 표기가 있다.
  - 원 내부에 여러 선분이 그려져 있다.
- 보기:
  - `( 선분 ㄱㄹ , 선분 ㄴㄹ , 선분 ㅇㄷ )`
- 정답 표기:
  - `(정답)선분 ㄴㄹ`
- 해설:
  - `(해설)원의 중심을 지나고 원 위의 두 점을 이은 선분을 찾아보면 선분 ㄴㄹ입니다.`

[수식/관계 구조]
- 등호를 포함한 계산식은 보이지 않는다.
- 수식 전개나 변형 연쇄도 없다.
- 대신 도형 관계가 핵심이다.
- 해설 문장에 나타난 관계는 다음처럼 읽힌다.
  - **원의 중심을 지난다**
  - **원 위의 두 점을 이은 선분이다**
  - 이 조건을 만족하는 선분을 찾는 문제이다.

[등호별 변환 의미]
- 해당 없음.
- 등호로 연결된 변환 과정이 없다.
- 계산식 변형이나 단계별 식 변화도 보이지 않는다.

[빈칸 정책]
- 빈칸 문제 형식이 아니다.
- 답을 추론해서 채우지 않는다.
- 원본에 **정답이 인쇄되어 보이는 경우**에만 관찰 사실로 적는다.
- 이 화면에서는 정답 표기 `선분 ㄴㄹ`이 실제로 보이므로, 이를 **정답 표기 사실**로만 기록한다.
- 추론으로 다른 보기의 정오를 새로 단정하지 않는다.

[반복 패턴]
- 보기가 모두 `선분 + 한글 자음 두 개` 형식으로 반복된다.
  - `선분 ㄱㄹ`
  - `선분 ㄴㄹ`
  - `선분 ㅇㄷ`
- 그림 안의 점/기호도 자음 하나씩으로 라벨링되어 반복된다.
  - `ㄱ`, `ㄴ`, `ㄷ`, `ㄹ`, `ㅇ`
- 같은 종류의 “선분” 표현이 여러 번 나와 선택형 구조를 만든다.
- 원 내부에 여러 선분이 겹쳐 보여, 후보들 중 하나를 고르는 시각 패턴이 강조된다.

[DSL 작성 힌트]
- 도형 구성은 **원 1개 + 중심 표시 + 둘레 점 여러 개 + 선분 여러 개 + 텍스트 영역**으로 나누는 것이 적절하다.
- 원의 중심은 작은 점과 함께 별도 라벨로 표시해야 한다.
  - 중심 부근의 표시는 `ㅇ`로 읽히는 형태이다.
- 원 둘레의 점/라벨은 다음 위치 관계로 배치된다.
  - 위쪽: `ㄱ`
  - 오른쪽: `ㄹ`
  - 왼쪽 아래: `ㄴ`
  - 아래쪽: `ㄷ`
- 원 내부 선분은 적어도 3개가 보인다.
  - `ㄱ`과 `ㄹ`을 잇는 선분
  - `ㄴ`과 `ㄹ`을 잇는 선분
  - 중심에서 `ㄷ` 쪽으로 내려가는 선분
- 보기 텍스트는 괄호를 포함한 한 줄 문자열로 배치하는 편이 자연스럽다.
- 정답 표기와 해설은 보기와 분리된 하단 텍스트 블록으로 두는 것이 좋다.
- 오른쪽 상단에 그림, 아래쪽에 보기와 정답/해설이 이어지는 세로 배치를 반영하면 원본과 유사하다.
- 선분과 문자 라벨이 겹치지 않도록 여백을 넉넉히 둬야 한다.
- 원 내부의 선분 끝점은 도형에 맞춰 자연스럽게 원 둘레 또는 중심 근처에 닿도록 처리한다.

[semantic/solvable 힌트]
- 이 문제는 **지름의 정의를 아는지**를 묻는 도형 인식 문제이다.
- 해설 문장에 따르면 기준은 다음 두 가지이다.
  1. 원의 중심을 지나야 함
  2. 원 위의 두 점을 이은 선분이어야 함
- 따라서 DSL 관점에서는 “원 + 중심점 + 후보 선분들”의 관계를 정확히 표현하는 것이 핵심이다.
- 정답 표기는 원본에 이미 보이므로, 문제 풀이를 새로 만들기보다 **보이는 정답 문구의 위치와 형식**을 유지하는 것이 중요하다.

[주의 및 TODO]
- `ㅇ`은 원의 중심 표식으로 보이지만, 문자 형태만 보면 `O`와 비슷하게 보일 수 있다. 위치 기준으로 중심 표식임을 처리하는 편이 안전하다.
- 원본 이미지에서 중심 표식은 작은 점과 문자 `ㅇ`이 함께 보인다. 두 요소를 분리해 인식할지, 하나의 중심 라벨로 묶을지 TODO로 남겨도 된다.
- 선분의 정확한 끝점은 해상도 영향으로 약간 애매할 수 있으므로 과도하게 단정하지 말 것.
- 정답 `선분 ㄴㄹ`은 하단에 인쇄되어 **보이는 사실**로만 기록하고, 추가 추론 설명은 피할 것.
- 색상 차이는 보이지만, DSL 작성 핵심은 도형과 텍스트 배치이므로 색상은 필수 요소인지 TODO로 검토해도 된다.
