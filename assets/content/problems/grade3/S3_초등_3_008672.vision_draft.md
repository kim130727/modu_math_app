# [세부 내용]
- 왼쪽 위의 `□ 52.`는 문제 번호 앞의 작은 네모 표시로 보입니다.
- 지시문 `알맞은 말을 선택해 보시오.`가 문제의 요구를 나타냅니다.
- 가운데 도형:
  - 큰 원 1개가 있습니다.
  - 원의 중심 부근에 작은 점이 있고 `O`라고 표시되어 있습니다.
  - 중심에서 원둘레로 뻗은 선분이 3개 있습니다.
  - 각 선분의 끝점은 원 위에 있으며, 각각 `ㄱ`, `ㄴ`, `ㄷ`으로 표시되어 있습니다.
- 도형에서 선분들은 서로 다른 방향으로 퍼져 있어, 중심에서 여러 점으로 연결된 구조가 분명하게 보입니다.
- 문장형 보기:
  - `선분 ㄱO, 선분 ㄴO, 선분 ㄷO은 모두 원의 (지름, 반지름, 중심)입니다.`
  - 괄호 안의 `지름, 반지름, 중심`은 선택해야 하는 말들로 보이며, 하나를 고르는 형태입니다.
- 아래 `(정답)반지름`은 인쇄된 해설/정답 표기입니다.
- 해설 문장은 중심과 원 위의 점을 잇는 선분의 성질을 설명합니다.

# [Python DSL 작성 힌트]
- 문제의 핵심 도형은 다음 요소로 분리해 그리면 좋습니다.
  - 원 1개
  - 중심점 `O`
  - 중심에서 원둘레로 향하는 선분 3개
  - 원둘레의 점 3개와 각각의 문자 `ㄱ`, `ㄴ`, `ㄷ`
- 텍스트는 상단 지시문, 중앙 설명문, 하단 정답/해설의 3개 영역으로 나누어 배치하는 것이 적절합니다.
- 보기 문장은 괄호를 포함해 그대로 배치하되, 답을 선택하는 부분은 비워 두지 말고 **문장 전체를 시각적으로만 재현**하는 방식이 좋습니다.
- 도형의 점과 문자 위치 관계를 유지해야 합니다:
  - `ㄱ`은 왼쪽 위 쪽 원둘레 근처
  - `ㄴ`은 아래쪽 원둘레 근처
  - `ㄷ`은 오른쪽 위 원둘레 근처
- 중심점 `O`는 원의 중심보다 약간 왼쪽 아래에 점처럼 표시되어 보일 수 있으므로, 실제 좌표 배치 시 시각적 중심과 표기 위치를 함께 고려합니다.
- 정답과 해설은 문제 본문과 분리된 하단 텍스트 블록으로 구성합니다.

# [주의할 점]
- 이 문제는 개념 확인 문제이므로, **답을 추론하여 빈칸에 직접 넣는 작업은 하지 않아야 합니다.**
- `지름`, `반지름`, `중심` 중 어느 것이 맞는지 여기서 판정하지 말고, 화면에 보이는 텍스트만 기록해야 합니다.
- 선분 이름에서 한글 자모(`ㄱ`, `ㄴ`, `ㄷ`)와 알파벳/기호 `O`의 조합을 정확히 구분해야 합니다.
- 해설의 내용은 결과 설명이므로, Python DSL에 포함할 때는 별도 텍스트 영역으로 처리하는 것이 좋습니다.
