# [전체적인 구성]
- 한 개의 초등 수학/도형 객관식 문제 화면이다.
- 상단에는 문제 번호와 문제 문장이 있고, 가운데에는 원 도형 그림이 배치되어 있다.
- 오른쪽에는 보기 3개가 세로로 나열되어 있으며, 각 보기는 동그라미 기호(①, ②, ③ 형태의 선택지 표시)와 함께 문장으로 제시된다.
- 하단 왼쪽에는 정답 표시가 있고, 그 아래에 해설 문장이 한 줄로 있다.
- 전체적으로 **문제 문장 → 도형 그림 → 선택지 목록 → 정답/해설**의 흐름으로 구성되어 있다.

# [보이는 텍스트]
- 상단 문제 문장:
  - `2. 원의 반지름을 나타내는 선분의 기호를 선택해 보세요.`
- 오른쪽 보기:
  - `① 선분 ㄱㄴ`
  - `② 선분 ㄷㄹ`
  - `③ 선분 ㅇㄹ`  
  - 다만 마지막 보기의 글자는 이미지상 선명도가 낮아 보이며, 실제로는 `ㅇㄹ`처럼 보이지만 정확한 판독은 약간 불확실하다.
- 하단 정답 표기:
  - `(정답) ③`
- 하단 해설:
  - `(해설) 원의 중심과 원 위의 한 점을 이은 선분은 선분 ㅇㄹ입니다.`
- 도형 안 문자:
  - 중심 부근에 `ㅇ`으로 보이는 표기
  - 원 위의 점들에 `ㄱ`, `ㄴ`, `ㄷ`, `ㄹ` 표기가 보임

# [반복되는 시각 패턴]
- 보기 항목은 모두 같은 형식으로 정렬되어 있다.
  - 동그라미로 된 번호 기호 + `선분` + 두 한글 자모 표기
- 도형 안에서도 점/선분에 대응하는 한글 기호가 여러 개 반복적으로 등장한다.
  - 원 위의 서로 다른 지점에 `ㄱ, ㄴ, ㄷ, ㄹ`
  - 중심으로 보이는 점에 `ㅇ`
- 선분 표현이 서로 다른 방향으로 그려져 있어, 같은 원 내부에서 여러 선분의 종류를 비교하도록 만든 구성이다.
- 정답과 해설은 문제 본문과 분리되어 아래쪽에 배치되어 있어, 학습용 설명 구조가 반복된다.

# [세부 내용]
- 가운데 그림은 원 하나를 크게 그린 뒤, 그 안에 여러 선분을 표시한 도형이다.
- 원 내부에는 다음과 같은 요소가 보인다.
  - 원의 중심으로 보이는 점 1개
  - 중심에서 오른쪽 원둘레의 점까지 이어진 가로 선분 1개
  - 왼쪽 위에서 아래로 내려오는 세로에 가까운 선분 1개
  - 왼쪽 아래에서 오른쪽 점으로 올라가는 대각선 선분 1개
- 각 점에는 한글 자모가 붙어 있어 선분 이름과 대응시키는 방식이다.
- 오른쪽 보기들은 도형의 각 선분 중 어떤 것이 반지름인지 묻는 선택지로 보인다.
- 하단 해설은 “원의 중심과 원 위의 한 점을 이은 선분”이라는 정의형 문장으로 구성되어 있다.
- 문장 끝의 `선분 ㅇㄹ입니다` 부분은 그림에서 특정 선분에 대응시키는 설명으로 읽힌다.
- 도형의 선분 중 하나는 중심에서 원둘레까지 이어져 있어, 보기 중 해당 선분을 찾도록 유도하는 구조다.  
  - 다만 이는 문제 의도에 대한 **의미적 해석**이고, 실제 답은 그대로 추론해 채우지 않도록 주의해야 한다.

# [Python DSL 작성 힌트]
- 문제를 구성할 때 다음 요소를 분리해 배치하면 좋다.
  1. 상단 문제 문장 영역
  2. 중앙 도형 영역
  3. 오른쪽 보기 영역
  4. 하단 정답/해설 영역
- 도형 영역에서는:
  - 원 1개
  - 중심 점 1개
  - 원 위의 점 여러 개
  - 서로 다른 방향의 선분 3개 정도
  를 별도 객체로 표현하면 좋다.
- 각 점 옆의 한글 기호는 도형 요소와 가까이 붙여 배치해야 한다.
- 보기 목록은 세로 정렬로, 각 줄의 시작 위치를 맞추는 방식이 적절하다.
- 해설 텍스트는 문제와 시각적으로 분리된 하단 문단으로 넣는 것이 좋다.
- 정답 표시는 별도 줄에 두어, 문제 본문과 혼동되지 않게 한다.

# [주의할 점]
- 답을 추론해서 빈칸에 넣지 말아야 한다. 이 화면은 선택형 문제이므로 빈칸 자체는 없지만, 정답을 분석 초안에 확정적으로 채워 넣는 방식은 피해야 한다.
- 도형 안의 문자 `ㅇ, ㄱ, ㄴ, ㄷ, ㄹ`는 선분 이름 표시에 쓰이는 기호이므로 숫자나 다른 문자로 오독하지 않도록 주의한다.
- 오른쪽 보기의 마지막 항목 글자는 이미지 해상도 때문에 약간 흐릿해 보일 수 있다.
- 하단 해설은 정답을 직접 밝히는 형태지만, 분석 초안에서는 이를 그대로 옮겨 적되 문제 풀이 과정으로 확장하지 않는다.
- `정답 ③`과 해설 문장이 함께 보이지만, 시각 분석에서는 “보인다” 수준으로만 기록하는 것이 안전하다.

# [불확실한 부분]
- 오른쪽 보기의 마지막 선분 표기가 `ㅇㄹ`인지, 또는 비슷한 모양의 다른 자모 조합인지 약간 불확실하다.
- 도형 내부의 점 표기 `ㅇ`이 정확히 원기호인지, 한글 자모 `ㅇ`인지 이미지상 구분이 다소 모호하다.
- 선분 이름으로 쓰인 한글 자모들이 화면상 작아 보여 위치 매칭은 대체로 가능하지만, 세부적인 정확도는 원본 이미지 해상도에 영향을 받는다.
