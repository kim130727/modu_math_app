# [전체적인 구성]
- 좌측 상단에 문제 번호와 지시문이 있고, 그 아래 가운데 영역에 **세 개의 원 그림이 가로로 나란히** 배치되어 있습니다.
- 가운데의 세 그림은 **연한 보라색 직사각형 테두리** 안에 들어 있습니다.
- 왼쪽 중간쯤에는 **(정답)** 표시와 함께 작은 원 그림 1개가 따로 있습니다.
- 그 아래에는 **(해설)** 문장 설명이 두 줄로 적혀 있고, 맨 아래에는 **3개의 예시 그림**이 다시 가로로 배치되어 있습니다.
- 전체적으로 “문장 문제 + 선택 그림 3개 + 정답 표시 + 해설 + 아래 예시 그림”의 구조입니다.

- `(정답)`
- `(해설)`
- `첫째, 셋째 그림은 원의 반지름은 다르고 원의 중심을 옮겨 가며 그린 것입니다.`
- `둘째 그림은 원의 중심은 같고 반지름만 다르게 하여 그린 것입니다.`

# [반복되는 시각 패턴]
- 위쪽 선택지 영역에는 **원 3개**가 반복되며, 모두 같은 크기의 외곽 테두리 박스 안에 정렬되어 있습니다.
- 각 선택지는 모두 **겹쳐진 원 도형**으로 보이며, 차이점은 원의 **중심 위치**와 **반지름 크기**입니다.
- 아래 해설용 예시 그림도 역시 **원 3개**가 반복되지만, 각 그림 아래에 **파란 점**으로 중심 위치가 표시되어 있습니다.
- (정답) 옆의 작은 그림은 **동심원 형태 3개**가 겹친 구조로 보입니다.

# [세부 내용]
- 상단 문제 문장은 “원의 중심을 옮기지 않고 반지만 다르게 하여 그린 것”을 찾는 문제입니다.
- 선택지는 총 3개이며, 모두 원 그림이지만 형태가 서로 다릅니다.
  - **왼쪽 선택지**: 큰 원 안에 좌우로 겹친 두 개의 작은 원이 들어 있는 모양입니다.
  - **가운데 선택지**: 중심이 같은 듯한 **동심원 형태**로 보이며, 바깥원과 안쪽 원들이 같은 중심을 공유합니다.
  - **오른쪽 선택지**: 큰 원 안에 아래쪽으로 중심이 치우친 원들이 겹쳐 있는 모양입니다.
- 가운데 선택지와 별도로 왼쪽에 `(정답)` 표시가 있는 작은 그림이 있으며, 이 그림도 **동심원 3개**처럼 보입니다.
- 아래 해설의 첫 문장은 첫째, 셋째 그림은 **중심이 이동한 그림**이라고 설명합니다.
- 아래 해설의 둘째 문장은 둘째 그림은 **중심은 같고 반지름만 다르게** 그린 것이라고 설명합니다.
- 하단의 3개 예시 그림에는 각 원의 중심을 나타내는 듯한 **파란 점**이 들어가 있습니다.
  - 왼쪽 예시: 파란 점 2개가 좌우로 떨어져 있어 중심이 다름을 시각적으로 보여줍니다.
  - 가운데 예시: 파란 점이 한 곳에 있어 중심이 같음을 보여줍니다.
  - 오른쪽 예시: 파란 점이 위아래로 분리되어 있어 중심이 이동했음을 나타내는 듯합니다.

# [Python DSL 작성 힌트]
- 상단 문제 번호와 문장은 **좌측 정렬 텍스트**로 배치합니다.
- 중앙 선택 영역은 **연한 보라색 사각형 테두리**를 먼저 그린 뒤, 그 안에 원 3개를 가로 정렬합니다.
- 각 선택지는 **원 여러 개가 겹친 도형**으로 표현해야 하며, 중심점이 같은지/다른지에 따라 위치를 달리 배치합니다.
- `(정답)` 표시는 좌측 중간 영역에 텍스트와 작은 원 그림을 함께 배치합니다.
- 해설 문장은 `(해설)` 라벨 뒤에 두 줄 텍스트로 넣습니다.
- 하단 예시 그림은 원 3개를 가로로 배치하고, 각 그림 안에 파란 점을 중심 표시처럼 추가합니다.
- 도형 요소를 그릴 때는:
  - 외곽 원,
  - 내부 원,
  - 중심점(파란 점),
  - 겹침 관계
  를 분리해서 작성하면 좋습니다.

# [주의할 점]
- 정답을 추론해 빈칸을 채우는 방식이 아닙니다. 이 문제는 **선택 그림을 고르는 시각 문제**입니다.
- `(정답)` 옆 그림이 실제 정답 표시인지, 또는 해설용 예시인지 문맥상 구분해 두는 것이 좋습니다.
- 아래 해설에 등장하는 파란 점은 **원 중심을 나타내는 시각적 표시**로 보이지만, 점의 의미를 단정적으로 확정하지는 않는 것이 좋습니다.
- 선택지의 원들은 모두 단순 원이 아니라 **겹쳐진 구조**이므로, 중심 좌표와 반지름을 각각 따로 관리해야 합니다.
- 보라색 테두리 박스는 선택지 전체를 감싸는 안내 영역처럼 보입니다.

# [불확실한 부분]
- `(정답)` 옆의 작은 원 그림이 실제 정답 표시인지, 별도 참고 그림인지 화면만으로는 완전히 단정하기 어렵습니다.
- 하단 예시의 파란 점이 정확히 “중심”을 뜻하는지 텍스트와 도형 맥락상 강하게 보이지만, 이미지 해상도상 세부 의미는 약간 불명확할 수 있습니다.
- 왼쪽/오른쪽 선택지 내부 원의 정확한 반지름 차이는 화면상 대략만 파악되며, 정밀한 수치까지는 알 수 없습니다.
