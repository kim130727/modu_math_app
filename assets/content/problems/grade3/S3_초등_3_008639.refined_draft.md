# [문제 개요]
- 초등 수학의 보기 선택형 문항이다.
- 중심에서 원둘레까지 뻗는 선분 3개가 그려진 원 그림을 보고, 반지름의 성질을 판단하게 하는 문제로 보인다.
- 상단에 문제 문장이 있고, 가운데에 그림, 아래에 선택 문장이 박스 안에 제시되며, 더 아래에 `(정답)`, `(해설)`이 인쇄되어 있다.

# [원본에 보이는 요소]
- 상단 문장:
  - `11. 원에 반지름을 3개 그은 것입니다. 반지름을 제어 알맞은 말을 선택하세요.`
  - 이미지상 `재어`는 `재어`로 보이지만, 문맥상 원본 판독은 `재어`가 아니라 `적어`일 가능성도 있어 보이나, 여기서는 원본에 보이는 형태를 우선 기록해야 함.
- 원 그림:
  - 화면 중앙 상단 쪽에 큰 원 1개가 있음.
  - 원 중심 부근에 작은 분홍색 점이 있음.
  - 중심 부근에 작은 흰색/빈 원처럼 보이는 표시가 있음.
  - 중심에서 원둘레로 향하는 선분 3개가 있음.
- 선택 문장 박스:
  - 연두색 테두리의 큰 직사각형 박스가 있음.
  - 박스 안 문장:
    - `한 원에서 원의 반지름들은 모두 ( 같습니다, 다릅니다 ).`
- 하단 텍스트:
  - `(정답)같습니다`
  - `(해설)반지름은 모두 같습니다.`
- 선분 색:
  - 왼쪽 방향에 가까운 하늘색
  - 오른쪽 위 방향의 연보라색
  - 아래 방향의 초록색

# [수식/관계 구조]
- 등호를 포함한 계산식은 없음.
- 그림-문장 대응 구조로 보임.
- 구조적으로는 다음 관계를 묻는 문항으로 읽힘:
  - 한 원의 반지름 여러 개
  - 이들의 길이 관계를 비교하여 `같습니다 / 다릅니다` 중 선택
- 단, 정답 판단은 새로 추론하지 않고, 원본 하단에 인쇄된 `(정답)같습니다`를 관찰 사실로만 기록한다.

# [등호별 변환 의미]
- 해당 없음.
- 등호 연쇄나 식 변형 과정이 보이지 않는다.
- 따라서 단계별 변환 설명 대상이 없다.

# [빈칸 정책]
- 이 문항에는 수치형 빈칸은 없고, 괄호 선택형 빈칸만 보인다.
- 괄호 안 선택지는 원본 표기 그대로 유지해야 하며, 빈칸을 추론값으로 채우지 않는다.
- DSL 작성 시에도 선택지의 원문 텍스트를 그대로 반영하고, 정답 선택 상태는 원본에 인쇄된 내용만 사실로 기록한다.
- 보기에 있는 `같습니다, 다릅니다` 중 어떤 것을 선택하는지 새로 판단해 채워 넣지 않는다.

# [반복 패턴]
- 중심에서 바깥으로 뻗는 선분 3개가 반복적으로 배치되어 있다.
- 세 선분은 서로 다른 방향으로 뻗지만, 모두 같은 도형 요소인 반지름 예시로 읽히는 반복 패턴이다.
- 선택지 구조도 반복적이다:
  - 한 문장 안에 두 개의 선택 항목 `( 같습니다, 다릅니다 )`가 같은 형식으로 들어 있다.
- 하단의 `(정답)`, `(해설)`도 서로 같은 접두 구조를 반복한다.

# [DSL 작성 힌트]
- 원 1개를 중심 기준으로 배치하고, 중심에서 원둘레까지 이어지는 선분 3개를 추가하는 구성이 핵심이다.
- 중심점은 작은 점 또는 작은 원으로 별도 표시하는 것이 적절하다.
- 선분은 각기 다른 색으로 표현되어 있으며, 방향도 서로 다르다.
- 상단 문제 문장은 왼쪽 정렬된 한 줄 텍스트로 배치한다.
- 원은 상단 중앙부에, 선택 문장 박스는 그 아래에 가로로 길게 배치한다.
- 선택 문장 박스는 연두색 테두리의 직사각형으로 표현하고, 내부 문장은 가운데 정렬에 가깝게 배치하는 것이 자연스럽다.
- 하단의 `(정답)`, `(해설)`은 박스 아래 왼쪽에 세로로 배치한다.
- 그림 요소와 텍스트 요소 사이에 충분한 세로 간격이 있다.
- 원 내부의 작은 흰색/빈 원과 분홍색 점은 중심 강조용 표시로 보이므로, 가능하면 별도 마커로 반영하되 과도하게 단정하지 않는다.

# [semantic/solvable 힌트]
- 보이는 문장은 반지름의 성질을 묻는다.
- 원 안의 3개 선분은 모두 중심에서 원둘레로 이어지는 형태이므로, 문제 의도상 같은 길이의 반지름들을 보여 주는 예시로 해석된다.
- 다만 이 항목은 해석 힌트일 뿐이며, 답을 새로 계산하거나 추론해서 확정하지 않는다.
- 원본 하단에는 `(정답)같습니다`, `(해설)반지름은 모두 같습니다.`가 인쇄되어 있어, 관찰 가능한 정답 표기가 존재한다.

# [주의 및 TODO]
- 상단 문장 판독에서 `제어/재어/적어`처럼 보이는 부분이 해상도상 다소 불명확하다. 실제 원본 이미지 기준으로 최종 판독 필요.
- 원 내부의 작은 `O` 모양 표시는 글자인지 중심 표시인지 애매하다.
- 분홍색 점의 정확한 역할은 확정할 수 없으므로, 중심 강조 표시 정도로만 다룬다.
- 선분의 끝점이 정확히 원둘레에 닿는지 여부는 해상도상 약간 모호하다.
- 정답 선택은 원본에 이미 인쇄된 `(정답)같습니다`만 사실로 기록하고, 추가 추론으로 보완하지 않는다.
