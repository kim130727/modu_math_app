# [전체적인 구성]
- 화면 상단 왼쪽에 문제 번호 **84.**가 있고, 그 오른쪽에 문제 문장이 한 줄로 배치되어 있습니다.
- 문제 문장 아래에는 연한 주황색 테두리의 **직사각형 선택 상자 3개**가 가로로 일정 간격을 두고 나열되어 있습니다.
- 각 선택 상자 안에는 **빈칸(작은 네모 칸)**과 나눗셈 기호가 함께 표시되어 있으며, 각 상자는 서로 다른 나누는 수와 짝을 이루는 형태입니다.
- 선택 상자 아래쪽 왼편에는 **(정답)** 표시와 함께 작은 네모 빈칸, 나눗셈 기호, 숫자가 이어진 **정답식**이 따로 있습니다.
- 가장 아래에는 **(해설)**이 붙은 설명 문장이 한 줄로 제시되어 있습니다.

# [보이는 텍스트]
- `84. 나머지가 6이 될 수 없는 식을 찾아 선택해 보세요.`
- 왼쪽 선택 상자: `□ ÷ 8`
- 가운데 선택 상자: `□ ÷ 6`
- 오른쪽 선택 상자: `□ ÷ 9`
- 아래 정답 표기: `(정답) □ ÷ 6`
- 아래 해설 문장: `(해설) 나누는 수가 6일 때, 나머지가 될 수 있는 수는 0, 1, 2, 3, 4, 5입니다.`

# [반복되는 시각 패턴]
- 가로로 나열된 선택 상자 3개가 같은 크기와 같은 테두리 스타일로 반복됩니다.
- 각 상자 안에는 모두 **빈칸 네모 + 나눗셈 기호 ÷ + 숫자**라는 동일한 구조가 반복됩니다.
- 숫자만 각각 **8, 6, 9**로 달라집니다.
- 아래 정답 영역도 `□ ÷ 6`처럼 상자 안 빈칸과 나눗셈 기호의 형식이 반복되어, 위의 선택지 형식과 닮아 있습니다.

# [수식 단계]
- 이 문제는 여러 단계의 등호 변형식이 보이는 형식은 아닙니다.
- 따라서 수식 변환 단계나 연쇄적인 등호 표시는 관찰되지 않습니다.

# [등호별 변환 의미]
- 해당 없음.
- 화면에는 등호를 포함한 변형 과정이 나타나지 않습니다.

# [세부 내용]
- 문제 문장은 **“나머지가 6이 될 수 없는 식”**을 찾는 유형입니다.
- 시각적으로는 선택지 3개 중에서 하나를 고르는 형태이며, 각 선택지는 같은 템플릿으로 구성되어 있습니다.
- 선택지의 왼쪽에는 작은 정사각형 빈칸이 있고, 그 뒤에 `÷` 기호, 마지막에 한 자리 수가 붙어 있습니다.
- 아래 정답 표기는 동일한 형식으로 다시 한 번 제시되며, 해설 문장에서는 **나누는 수가 6일 때 가능한 나머지의 범위**를 글로 설명합니다.
- 해설 문장은 숫자 목록을 쉼표로 나열한 형태이며, 끝은 `5입니다.`로 마무리됩니다.
- 문장 전체가 초등 수학의 나눗셈-나머지 개념을 묻는 문제로 보이지만, 여기서는 문제 해결은 하지 않습니다.

# [Python DSL 작성 힌트]
- 상단 문제 번호와 문제 문장을 한 줄 텍스트로 배치합니다.
- 그 아래에 **3개의 주황색 테두리 박스**를 가로 정렬로 둡니다.
- 각 박스 내부에는 중앙 정렬에 가깝게 `□ ÷ 숫자` 형태를 넣습니다.
- 선택지의 숫자는 각각 `8`, `6`, `9`로 보이므로, 박스별 텍스트를 분리해서 입력하면 좋습니다.
- 아래쪽에는 `(정답)` 라벨과 함께 `□ ÷ 6` 형식의 줄을 따로 둡니다.
- 맨 아래 해설 문장은 라벨 `(해설)`과 함께 일반 텍스트로 배치합니다.
- 박스는 연한 주황색 채움 없이 **테두리만 있는 직사각형**으로 표현하는 것이 시각적으로 유사합니다.
- 빈칸은 실제 답을 넣지 말고 **사각형 기호 또는 placeholder**로 유지합니다.

# [주의할 점]
- 정답을 시각 분석 단계에서 추론해 채우지 마세요.
- 선택지의 빈칸은 숫자가 들어갈 자리이지만, 실제 값은 적지 마세요.
- 해설 문장에 포함된 수들의 의미를 계산해 확장 설명하지 마세요.
- 문제 해결 과정이 아니라 **화면 구성과 텍스트 배치**에 집중해야 합니다.
- 정답 영역이 보이더라도, 이를 실제 해답으로 확정해 쓰지 말고 **보이는 형태만** 기록하세요.

# [불확실한 부분]
- 선택 상자 내부의 빈칸이 정확히 어떤 폰트/도형으로 그려졌는지는 이미지 해상도상 세부가 다소 अस्प겠습니다. 다만 작은 사각형 빈칸으로 보입니다.
- `(정답)` 줄의 빈칸이 선택지와 동일한 크기인지, 약간 다른 크기인지는 이미지상 완전히 확정하기 어렵습니다.
- 박스 테두리 색은 연한 주황색 계열로 보이지만, 정확한 색상 코드까지는 판별하지 않았습니다.
