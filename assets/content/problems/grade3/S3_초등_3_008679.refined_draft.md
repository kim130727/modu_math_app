[문제 개요]
- 문제 번호는 `61.`로 보인다.
- 문장형 지시문이 상단에 2줄로 배치되어 있다.
- 중심 그림은 왼쪽 도형에서 오른쪽 도형으로 바뀌는 과정을 화살표로 보여 준다.
- 아래에는 선택형 문장이 1줄로 제시되고, 그 아래에 `(해설)` 설명 문장과 맨 아래 짧은 답 표기처럼 보이는 텍스트가 있다.
- 주제는 원을 반으로 접었을 때 생기는 선과 원의 지름의 관계를 묻는 내용으로 보인다.

[원본에 보이는 요소]
- `61.`
- `원 모양 종이를 똑같이 둘로 나누어지도록 반을 접었다가 폈더니 선이 생겼습니다. 알맞은 말을 선택하세요.`
- 그림:
  - 왼쪽: 연한 분홍색 테두리의 반원형 도형처럼 보이는 그림
  - 가운데: 오른쪽 방향 화살표
  - 오른쪽: 연한 분홍색 테두리의 원형 도형 안에 가로 점선
- 선택형 문장:
  - `(2) 원의 지름은 원을 똑같이 ( 둘, 넷 )(○)로 나눕니다.`
- 해설:
  - `(해설)반을 접어 생긴 선분은 원의 중심을 지나는 원의 지름입니다. 원의 지름은 원을 똑같이 둘로 나눕니다.`
- 맨 아래:
  - `(2) 둘`

[수식/관계 구조]
- 등호 `=`를 이용한 수식 변형은 보이지 않는다.
- 대신 다음과 같은 개념 관계가 텍스트로 제시된다.
  - 반을 접어 생긴 선분
  - 원의 중심을 지나는 선분
  - 원의 지름
  - 원을 똑같이 둘로 나눔
- 그림의 구조는 `왼쪽 도형 → 오른쪽 도형`의 변화 관계로 읽힌다.
- 선택지 구조는 `( 둘, 넷 )` 중에서 하나를 고르는 형태로 보인다.

[등호별 변환 의미]
- 해당 없음: 등호 연쇄나 식 변형은 보이지 않는다.
- 다만 해설 문장 안에서 의미 연결은 다음처럼 읽힌다.
  - `반을 접어 생긴 선분` → `원의 중심을 지나는 선분`
  - `원의 중심을 지나는 선분` → `원의 지름`
  - `원의 지름` → `원을 똑같이 둘로 나눔`
- 그림의 변환 의미는 다음처럼 보인다.
  - 왼쪽은 종이를 반으로 접은 상태를 단순화한 도형처럼 보인다.
  - 오른쪽은 펼쳤을 때 원 안에 선이 생긴 상태를 나타낸다.
  - 가운데 화살표는 접은 상태에서 펼친 상태로의 변화를 나타낸다.

[빈칸 정책]
- 본문에는 빈칸형 선택 구조가 있으나, 추론으로 채우지 않는다.
- `( 둘, 넷 )` 안의 정답은 원본에 선택된 표시가 보이더라도, 확정적으로 채워 넣지 말고 빈칸 상태로 유지한다.
- 하단의 `(2) 둘`도 정답 확정값으로 단정하지 말고, 원본에 보이는 문자열 그대로만 다룬다.
- DSL 작성 시에도 빈칸은 추론값 대신 빈칸 슬롯으로 처리해야 한다.

[반복 패턴]
- `(2)` 표기가 두 번 등장한다.
  - 한 번은 선택형 문장 앞
  - 한 번은 맨 아래 짧은 표기
- `원의 지름은 원을 똑같이 둘로 나눕니다.`라는 뜻의 문장이 본문 선택 문장과 해설에서 반복된다.
- `원을 똑같이 둘로`라는 표현이 해설과 선택 문장에 모두 나타난다.
- 그림은 `접은 상태 → 화살표 → 펼친 상태`의 시각적 반복 구조를 가진다.

[DSL 작성 힌트]
- 상단 텍스트, 중앙 그림, 하단 선택 문장, 해설, 맨 아래 짧은 표기를 서로 다른 블록으로 분리하는 구성이 적절하다.
- 그림 배치는 가로 정렬이 자연스럽다.
  - 왼쪽: 반원형/접힌 상태 도형
  - 가운데: 오른쪽 방향 화살표
  - 오른쪽: 원형 도형과 내부 점선
- 오른쪽 원 내부의 가로 점선은 별도 선분 요소로 분리하면 좋다.
- 왼쪽 도형의 위쪽에 겹쳐 보이는 얇은 선은 접힌 흔적처럼 보이므로, 필요하면 보조 선으로 처리할 수 있다.
- 선택형 문장에서는 `( 둘, 넷 )`와 `○`가 같은 줄에 붙어 있으므로, 선택지 묶음과 선택 표시를 함께 배치하는 것이 좋다.
- `(해설)`은 본문과 시각적으로 구분되는 설명 블록으로 처리하는 것이 자연스럽다.
- 하단의 `(2) 둘`은 별도 짧은 텍스트 블록으로 분리해 두는 편이 안전하다.

[semantic/solvable 힌트]
- 보이는 문장만 기준으로 하면, 문제는 원의 지름이 무엇을 하는지에 대한 개념 확인형이다.
- 해설에서 `원의 지름은 원의 중심을 지나는 원의 지름입니다.`라는 식의 설명이 주어진다.
- 원의 지름이 원을 둘로 나눈다는 의미가 텍스트로 직접 제시되어 있다.
- 다만 정답을 새로 계산하거나 추론할 필요는 없고, 원본에 보이는 선택 구조와 해설만 반영하면 된다.
