# 시각 분석 초안

[전체적인 구성]  
- 화면 상단에는 문제 번호와 지시문이 있고, 가운데에는 원 그림, 그 아래에는 표, 그리고 선택형 문장이 이어진다.  
- 하단에는 “(정답)”과 “(해설)”로 보이는 텍스트가 별도로 배치되어 있다.  
- 전체적으로 **도형 관찰 → 표 확인 → 문장 선택** 흐름의 구성이다.

[보이는 텍스트]  
- 상단 문제 문장:  
  - `17. 지름을 나타내는 선분을 찾아 길이를 잰 값을 보고, 맞는 말을 선택해 보세요.`  
- 원 주변의 글자:  
  - `ㄱ, ㄴ, ㄷ, ㄹ, ㅁ, ㅂ, ㅅ, ㅇ`처럼 보이는 한글 자모가 원 둘레와 중심 부근에 배치되어 있다.  
  - 중심 근처에 `ㅇ`처럼 보이는 글자가 있다.  
- 표의 글자:  
  - 첫 행: `지름`, `선분 ㄱㅁ`, `선분 ㄴㅂ`, `선분 ㄹㅅ`  
  - 둘째 행: `길이(cm)`, `3`, `3`, `3`  
- 선택 문장:  
  - `한 원에서 원의 지름은 모두 (같습니다, 다릅니다).`  
- 하단 텍스트:  
  - `(정답) 같습니다`  
  - `(해설) 선분 ㄱㅁ, 선분 ㄴㅂ, 선분 ㄹㅅ은 모두 원 위의 두 점을 이은 선분이고 원의 중심을 지나므로 지름입니다. 한 원에서 원의 지름은 모두 같습니다.`

[반복되는 시각 패턴]  
- 원 안에 여러 개의 선분이 **중심을 기준으로 방사형**으로 그려져 있다.  
- 표에서는 지름 후보가 여러 개 나열되고, 각 길이 값이 **모두 3**으로 반복되어 있다.  
- 선택 문장은 괄호 안에 두 개의 선택지 `같습니다, 다릅니다`가 나란히 제시된 형태이다.

[수식 단계] (해당될 때)  
- 별도의 등호를 포함한 수식 변형 과정은 보이지 않는다.  
- 표에서 각 선분의 길이 값 `3`이 반복 제시되는 정도만 보인다.

[등호별 변환 의미] (해당될 때)  
- 해당 없음. 등호를 가진 계산식이나 변형식이 보이지 않는다.

[세부 내용]  
- 원 그림은 중심이 표시된 원이며, 중심 점이 `ㅇ`처럼 보이는 문자와 함께 표시되어 있다.  
- 원 둘레에는 여러 점이 있고, 그 점들을 연결한 선분들이 원 내부를 가로지른다.  
- 특히 표에 제시된 `선분 ㄱㅁ`, `선분 ㄴㅂ`, `선분 ㄹㅅ`은 각각 길이가 `3cm`로 적혀 있다.  
- 이 선분들은 시각적으로 원의 중심을 지나도록 그려진 것으로 보인다.  
- 선택 문장은 괄호 안의 두 단어 중 하나를 고르는 형식이다.  
- 하단의 정답/해설 텍스트는 문제를 푼 결과와 설명으로 보이지만, DSL 작성 시에는 **문제 본문 요소와 구분**해서 다루는 것이 좋다.

[Python DSL 작성 힌트]  
- 상단 문제 번호와 지시문을 텍스트 객체로 배치한다.  
- 가운데에 원 1개를 그리고, 중심 점을 표시한다.  
- 원 둘레의 여러 위치에 자모 라벨을 배치하고, 중심을 지나는 선분들을 여러 개 그린다.  
- 표는 2행 구조로 만들고, 첫 행은 `지름 / 선분 ㄱㅁ / 선분 ㄴㅂ / 선분 ㄹㅅ`, 둘째 행은 `길이(cm) / 3 / 3 / 3`로 보이도록 구성한다.  
- 선택 문장은 괄호가 포함된 한 줄 텍스트로 배치한다.  
- 하단의 `(정답)`, `(해설)` 텍스트는 문제 이미지의 일부이므로 필요 시 별도 텍스트 영역으로 넣되, 정답을 새로 계산해 쓰지 말고 보이는 그대로 옮긴다.

[주의할 점]  
- 지름이 되는 선분의 이름을 새로 추론해서 추가하지 않는다.  
- 선택지 중 정답을 임의로 확정하지 않는다.  
- 원 주변 자모의 정확한 위치는 이미지에서 다소 작게 보이므로, 배치 시 상대 위치를 참고 수준으로만 반영한다.  
- 표의 테두리 색과 선 굵기는 약간 연한 초록색 계열로 보이지만, 색상은 구현 환경에 따라 단순화해도 된다.  
- 하단의 해설 텍스트는 본문 문제와 분리된 영역처럼 보여서, 문제 재현 목적에 따라 포함 여부를 구분해야 한다.

[불확실한 부분]  
- 원 둘레의 자모 중 일부는 해상도상 정확히 구분이 어렵다.  
- 중심에 표시된 문자 `ㅇ`인지, 단순한 중심표시인지가 완전히 명확하지 않다.  
- 표의 셀 경계선과 글자 정렬은 실제 이미지에서 미세하게 비대칭일 수 있다.  
- 하단의 `(정답) 같습니다`와 `(해설)`은 문제 본문인지 별도 해설 영역인지 애매할 수 있으므로, DSL 작성 시 요구사항에 따라 분리 처리 여부를 결정해야 한다.
