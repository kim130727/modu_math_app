[문제 개요]
- 문제 번호: `36.`
- 질문 문장: `길이가 가장 긴 선분은 어느 것인가요?`
- 문제 유형: 도형을 보고 보기 중 맞는 선분을 고르는 선택형 문제
- 원본 이미지 기준으로, 하단에 `(정답) ②`와 해설 문장이 함께 인쇄되어 있음
- 정답/해설은 **보이는 사실**로만 기록하고, 별도 계산으로 새로 구한 값처럼 쓰지 않음

[원본에 보이는 요소]
- 왼쪽 위에 작은 네모 모양 기호 `□`와 문제 번호 `36.`이 보임
- 문제 문장은 왼쪽 상단에 가로로 배치됨
- 오른쪽 위에는 큰 원 도형이 있음
- 원 안에는 위에서 아래로 거의 평행한 가로 선분 4개가 그려져 있음
- 각 선분의 양 끝에는 한글 자모가 붙어 있음
  - 맨 위 선분: 왼쪽 `ㄱ`, 오른쪽 `ㄴ`
  - 두 번째 선분: 왼쪽 `ㄷ`, 오른쪽 `ㄹ`
  - 세 번째 선분: 왼쪽 `ㅁ`, 오른쪽 `ㅂ`
  - 맨 아래 선분: 왼쪽 `스`, 오른쪽 `ㅅ`
- 원의 중앙 부근에 작은 원/점 표시가 있음
- 보기 4개가 아래쪽에 2열 2행으로 배치되어 있음
  - `① 선분 ㄱㄴ`
  - `② 선분 ㄷㄹ`
  - `③ 선분 ㅁㅂ`
  - `④ 선분 스ㅅ`
- 하단 왼쪽에 `(정답) ②`가 보임
- 그 아래에 `(해설)중심을 지나는 선분 ㄷㄹ이 길이가 가장 긴 선분입니다.`가 한 줄로 보임

[수식/관계 구조]
- 계산식이나 등식은 없음
- 도형-라벨 대응 구조만 있음
- 관계 구조를 문자로 정리하면 다음과 같음
  - 선분 후보 4개가 있음
  - 그중 원의 중심을 지나는 선분이 `ㄷㄹ`로 보임
  - 해설 문장에는 이 `ㄷㄹ`이 가장 길다고 인쇄되어 있음
- **보이는 사실**과 **수학적 의미**를 분리하면:
  - 보이는 사실: 원 안에 4개의 가로 선분이 있고, 각 선분에 자모가 붙어 있음
  - 추론되는 수학적 의미: 중심을 지나는 선분이 원에서 가장 길다는 성질을 묻는 문제로 보임
- 단, 이 성질은 해설에 인쇄되어 있는 문장을 근거로 한 관찰 수준으로만 기록하고, 새로 증명하거나 계산하지 않음

[등호별 변환 의미]
- 해당 없음
- 등호 연쇄, 식의 변형, 분수 변환은 보이지 않음
- 변환 단계 설명 필요 없음

[빈칸 정책]
- 이 문제에는 채워야 하는 계산 빈칸이 보이지 않음
- 선택형 보기의 번호와 선분 표기는 **보이는 그대로 유지**
- 해설이나 정답이 인쇄되어 있어도, 다른 빈칸을 추론값으로 채우지 않음
- 애매한 라벨이나 기호는 추정해서 바꾸지 말고 원문 그대로 또는 유보적으로 기록
- 특히 중앙의 작은 표시와 `스`/`ㅅ`은 확정하지 못하면 TODO로 남김

[반복 패턴]
- 원 안에 가로 선분 4개가 위에서 아래로 반복 배치되어 있음
- 각 선분은 왼쪽 끝 글자 + 선분 + 오른쪽 끝 글자 구조를 가짐
- 보기 문항도 공통 접두어 `선분` 뒤에 서로 다른 두 글자 조합이 반복됨
- 선택지 배치는 좌우 2열, 각 열에 2개씩 배치된 형태
- 원 내부의 선분들은 서로 거의 평행하며, 위아래 간격이 일정하지 않게 배치되어 있음
- 중앙 근처의 작은 원/점은 강조 표시처럼 반복 패턴과는 별도로 배치된 요소임

[DSL 작성 힌트]
- 문제 본문 블록과 도형 블록, 선택지 블록, 하단 정답/해설 블록을 분리해 두는 구성이 적절함
- 도형 구성 요소를 다음처럼 분해해서 표현하기 좋음
  - 큰 원 1개
  - 원 내부의 수평 선분 4개
  - 선분 양끝의 한글 라벨 8개
  - 중심 표시용 작은 원/점 1개
- 원 내부 선분들은 서로 평행한 선으로 처리하고, 길이는 원의 좌우 경계까지 닿는 형태로 보이게 하는 것이 자연스러움
- 선분별 세로 위치 차이를 드러내기 위해 위아래 간격을 조금씩 달리 배치
- 중심 표시가 선분 위에 포개져 있으므로, 도형 레이어에서 선분과 표시의 겹침 관계를 고려
- 보기 영역은 2열 2행 그리드로 정렬
- 하단의 `(정답) ②`, 해설 문장은 작은 글씨나 별도 주석 블록처럼 따로 배치
- 문제 번호 앞의 `□`는 체크박스처럼 보이므로, 필요하면 시각 요소로만 취급하고 의미는 단정하지 않음

[semantic/solvable 힌트]
- 이 문제의 핵심은 도형에서 **중심을 지나는 선분**을 찾는 것처럼 보임
- 해설 문장에 `중심을 지나는 선분 ㄷㄹ`이 가장 길다고 인쇄되어 있음
- 다만 DSL 작성용으로는 **정답을 새로 계산하기보다**, 원본에 인쇄된 정답/해설 사실만 참조하는 편이 안전함
- 도형 비교 의미는 다음처럼 요약 가능
  - 위쪽 선분, 가운데 선분, 아래 선분, 맨 아래 선분의 상대적 위치를 구분
  - 가운데 선분이 중심을 지난다는 점을 시각적으로 강조
- 정답이 원본에 인쇄되어 있으나, 추론값을 빈칸에 대입하는 방식은 사용하지 않음
